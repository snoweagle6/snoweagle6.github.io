下载安装Git、Node.js和Hexo

小白不用担心，接下来我们一起来学习

Git 简介、下载、安装

简介

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。
Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

下载

点击Git官网进行下载，默认选择64位。如果你需要使用32位，点击取消，点击32位进行下载即可。

安装

因为 Git本身是运行在Linux下的，所以要想在Windows环境下使用，必须构建一个Linux虚拟的环境。

  • 安装下载好的Git.exe，路点击next直至到Finish。
  • 进入Git的安装目录，例如“C:\Program Files\Git”，运行 git-bash.exe 。
  • git 访问远端仓库需要使用SSH-KEY进行认证。此时可以让管理员提供一个KEY，或者自己使用git-bash生成一个key。
    执行命令 ssh-keygen.exe -t rsa 然后有提示的地方就选择回车

    此时 C:\Users\hph28.ssh 多了两个问价id_rsa私钥,id_rsa.pub公钥。将id_rsa.pub重命名为 你的名字.pub 例如：hph.pub 并将其发送给管理员。
  • 配置用户信息
    初始化一下服务器的 git 用户，在任何一台客户端机器上使用 git，不管是自己生成KEY还是使用管理员提供的KEY，第一次都需要初始化一下：
    git config –global user.name “name” 建议使用自己名字的汉语拼音
    git config –global user.email “name@server”

Node.js 简介、下载、安装

简介

Node.js是一个能够在服务器端运行JavaScript的开放源代码、跨平台JavaScript运行环境。
Node.js大部分基本模块都用JavaScript语言编写。
后文的npm开头的命令都需要依赖Node.js。

下载

直接到Node.js的官方网站下载安装，看到最大的那两个绿色按钮，下载v6的就行。

安装

下载下来以后一路next下一步就可以了。

Hexo 简介、下载、安装

hexo 简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

hexo 下载、安装

打开你的 Git Bash 或者 Git Shell

一般安装好后 Git Bash 以后，右键任何地方，都会看到 Bash 和 GUI （图形界面）。
直接进入

依次执行以下指令：
  1. 安装hexo
    npm install hexo-cli -g
  2. 安装hexo-deployer-git（用来把博客部署到git上）
    npm install hexo-deployer-git --save

用hexo在本地搭建博客

初步搭建

  1. 创建放置博客的文件夹，可以在任何路径下（不建议中文路径，以免碰到不必要的错误）任意命名。
  2. 在该文件夹中，点击鼠标右键–> Git Bash
    • 执行 hexo init Hexo会自动下载并初始化搭建博客所需的所有文件。
    • 看到 INFO Start blogging with Hexo!，说明你的博客已经初步搭建好了。
    • 安装依赖包 npm install ,等待。。。。。没有看到ERROR字样，说明成功了

查看一下初步搭建结果

同样在你创建的文件夹下，执行已下命令：
生成静态文件
hexo g
启动本地服务，进行预览（这时只有本机能够访问博客）
hexo s
当看到Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.时，说明你搭建的博初次成型了。
